🔐 Tela de Login

Uma interface de autenticação dinâmica desenvolvida para um portal musical. O projeto foca em validação de formulários e design responsivo utilizando CSS moderno (Flexbox e Gradients).

🚀 Funcionalidades

Validação de Credenciais: Sistema de verificação lógica que valida se o e-mail e a senha coincidem com os dados cadastrados.

Feedback de Erro/Sucesso: Mensagens dinâmicas que informam ao usuário o status da tentativa de login.

Auto-Reset: Limpeza automática dos campos de entrada (input.value = '') após uma tentativa incorreta, melhorando a fluidez da experiência.

Layout Split Screen: Design dividido entre imagem e formulário, muito comum em aplicações SaaS modernas.

🧠 Lógica e Refinamento Técnico

Este projeto destaca habilidades fundamentais de front-end:

Manipulação do DOM: Uso intensivo de document.getElementById e querySelector para capturar e alterar elementos da página.

Controle de Fluxo: Implementação de condicionais if/else para gerenciar a lógica de permissão de acesso.

CSS Moderno:

Flexbox: Alinhamento centralizado e distribuição de elementos na tela.

Transparência e Sombras: Uso de rgba para fundos semitransparentes e box-shadow para profundidade visual.

Gradients: Fundo estilizado com linear-gradient para um visual vibrante.

🛠️ Tecnologias

JavaScript (ES6+): Lógica de autenticação e manipulação de classes.

HTML5: Estruturação de formulários e elementos semânticos.

CSS3: Estilização avançada, tipografia via Google Fonts (Lato) e efeitos de :hover.
